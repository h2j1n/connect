관리자 가이드

Admin 소개 네이버웍스 Admin은 관리자가 구성원 및 보안, 환경설정 등 전반적인 회사 계정을 관리할 수 있는 운영 도구입니다. IT 전문가가 아닌 관리자도 쉽게 사용할 수 있습니다. 관리자의 역할 관리자란 네이버웍스 회사 계정을 개설한 사용자며, 나중에 변경하거나 추가할 수 있습니다. Admin은 관리자만 접근할 수 있으며, 각 회사의 상황에 맞게 회사 계정과 서비스 환경을 설정할 수 있습니다. 내가 관리자가 맞긴 한데, 무엇부터 해야 할지 고민이신가요? 시작하기에서 step-by-step 가이드를 확인해보세요. Admin에서 무엇을 할 수 있나요? # 기본 환경 설정 회사 정보에서 회사의 기본 정보를 수정할 수 있습니다. 또한 회사 로고와 색상을 설정하여 조직의 소속감을 높일 수 있습니다. # 조직 및 구성원 관리 네이버웍스를 처음 사용하거나, 새로운 조직 또는 구성원이 생겼을 때 조직도에서 조직을 추가하거나 구성원을 추가할 수 있습니다. 또한, 필요에 따라 조직 삭제와 구성원 삭제도 가능합니다. # 서비스 관리 메시지 외부 연동 설정, 게시판의 파일 용량 제한, 메일의 파일 다운로드 기간 설정 등 서비스별 세부 설정을 관리할 수 있습니다. # 보안 관리 회사 정보를 안전하게 관리할 수 있도록 다양한 보안 기능을 설정할 수 있습니다. 구성원의 접속 현황과 활동 내역을 확인할 수 있고 필요 시 적절한 조치를 취할 수도 있습니다. 회사 정보에 접속한 기기를 확인할 수 있고, 메일, 메시지, 드라이브의 특정 활동을 모니터링하여 정보 유출을 예방할 수 있습니다.
회사 정보를 조회하고 수정합니다. 기본 정보 수정 네이버웍스를 이용하는 기업이나 단체의 이름과 전화번호를 확인하거나 추가할 수 있습니다. 이미 등록된 정보를 수정할 수 있습니다. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '회사 정보'를 눌러 '회사 정보' 화면으로 이동합니다. 모바일에서는 버튼을 탭하면 메뉴가 나타납니다. '기본 정보'로 이동합니다. 도메인 또는 그룹명 변경 최초 신청 시 입력한 그룹명 또는 도메인명입니다. 신청에 사용한 도메인 또는 그룹명을 변경할 수 있습니다. 도메인 변경하기 도메인을 사용하여 네이버웍스를 신청한 경우, 최고관리자는 도메인을 변경할 수 있습니다. '변경'을 눌러 '도메인' 메뉴로 이동하면, 등록한 보조 도메인을 기본 도메인으로 전환할 수 있습니다. 도메인을 전환하면, 모든 구성원의 계정이 'ID@(전환한 도메인)'으로 변경됩니다. 그룹명 변경하기 그룹명을 사용하여 네이버웍스를 신청한 경우, 최고관리자는 그룹명을 변경할 수 있습니다. 그룹명을 변경하면 변경 후 24시간 내에는 그룹명을 다시 변경할 수 없습니다. 또, 모든 구성원이 서비스에서 강제 로그아웃되며, 새로운 그룹명이 포함된 ID로 로그인을 해야 합니다. 구성원의 개인 이메일을 구성원 정보에 등록해 두면 개인 이메일로 변경사실이 통지되어 구성원에게 변경 사실을 공유할 때 편리합니다. 그룹명 오른쪽의 '변경'을 누릅니다. 주의사항을 체크하고, 변경할 그룹명을 입력 후 '그룹명 변경'을 누릅니다. 그룹명은 2~22자의 영문 소문자나 숫자, 하이픈(-)의 조합으로 구성해야 하며, 이미 사용 중인 그룹명은 사용할 수 없습니다. 또한, 'worksmobile', 'lineworks'를 포함한 경우와 'line', 'admin'은 그룹명으로 사용할 수 없습니다. 주의사항을 확인하고 '그룹명 변경'을 누릅니다. '확인'을 누릅니다. 그룹명 변경이 완료되면 최고 관리자의 개인이메일로 메일이 발송되며, 등록한 휴대폰 번호로 SMS가 전송됩니다. 기업/단체명 입력 기업이나 단체의 이름을 입력합니다. 화면 하단의 '고급 설정' 내 다국어 설정을 사용하고 적용 정보에서 '기업/단체명'을 선택하면 설정한 언어로 기업/단체명을 추가 입력할 수 있습니다. '기본 정보' 영역의 '기업/단체명' 항목에 회사명을 입력합니다. 1글자 이상 40자까지 입력 가능 일부 특수문자의 경우 사용 불가 대소문자 구별 없이 LINE, NAVER WORKS, WORKS MOBILE, NAVER, NAVER WORKS, NAVER CLOUD 사용 불가 오른쪽 상단의 '저장'을 누릅니다. 전화번호 입력 회사의 대표 전화번호를 입력합니다. '기본 정보' 영역의 '전화 번호' 항목에 회사의 대표 전화번호를 입력합니다. 1글자 이상 30자까지 입력 가능 숫자, -, *, +, P, T, (, ) 입력 가능 오른쪽 상단의 '저장'을 누릅니다. 언어·시간대/이용지역 설정 서비스 화면에 기본으로 적용되는 언어와 시간대를 설정합니다. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '회사 정보'를 눌러 '회사 정보' 화면으로 이동합니다. 모바일에서는 버튼을 탭하면 메뉴가 나타납니다. '언어·시간대/이용지역'으로 이동합니다. 언어 선택 회사의 구성원이 처음 접속했을 때 PC웹 서비스 화면에 기본적으로 노출될 언어를 선택합니다. 한국어가 기본 언어로 지정되어 있습니다. 단, 모바일앱과 PC앱은 구성원이 디바이스에서 설정한 언어가 노출됩니다. '언어·시간대/이용지역' 영역의 '언어' 항목에 서비스에 기본적으로 표현될 언어를 선택합니다. 한국어, 영어, 일본어, 중국어 간체, 중국어 번체 중 선택 가능 오른쪽 상단의 '저장'을 누릅니다. 구성원 정보 수정에서 특정 구성원의 언어를 변경하면 해당 구성원에게는 변경된 언어가 PC웹 서비스 화면에 노출됩니다. 시간대/이용지역 선택 서비스를 사용하는 지역과 시간대를 선택합니다. '서울, 대한민국 (GMT+09:00)'이 기본 시간대로 지정되어 있습니다. '언어·시간대/이용지역' 영역의 '시간대/이용지역' 항목에 서비스에 기본적으로 적용될 시간대를 검색하여 선택합니다. 국가명 또는 도시명을 한글로 검색할 수 있습니다. 오른쪽 상단의 '저장'을 누릅니다. 공식 계정 알림 수신 설정 이 기능은 네이버웍스 코어 상품의 Lite, Standard, Standard Plus 플랜 또는 웍스 드라이브 상품 또는 웍스 결재 상품 신청 후 사용 가능합니다. 네이버웍스 공식 계정의 알림 메시지 수신 여부를 설정합니다. 네이버웍스 공식 계정의 알림은 네이버웍스 서비스의 업데이트 등 소식을 전달합니다. Admin 왼쪽 메뉴에서 '기본환경'을 선택해 메뉴를 펼친 후 '회사 정보'를 눌러 '회사 정보' 화면으로 이동합니다. 모바일에서는 버튼을 탭하면 메뉴가 나타납니다. '공식 계정 알림'으로 이동합니다. 수신 여부 설정 네이버웍스 공식 계정의 알림 메시지 수신 여부를 설정합니다. '공식 계정 알림' 영역의 '알림 설정' 항목에 수신 여부를 선택합니다. 오른쪽 상단의 '저장'을 누릅니다. 수신 범위 설정 네이버웍스 공식 계정의 알림을 받기로 했을 때 받는 사람의 범위를 설정합니다. '공식 계정 알림' 영역의 '수신 범위' 항목에 수신 범위를 선택합니다. 관리자만 수신: 관리 권한이 있는 모든 관리자만 알림을 받습니다. 구성원(관리자 포함) 수신: 관리자 포함 회사의 모든 구성원이 알림을 받습니다. 오른쪽 상단의 '저장'을 누릅니다.
